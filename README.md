# AI Model Test | Static | YOLO

Легковесный GUI-инструмент для визуальной проверки и отладки моделей компьютерного зрения (YOLO) на статических изображениях. Позволяет быстро просматривать результаты детекции, оценивать уверенность модели и навигировать по набору данных.

## Возможности

*   **Визуализация детекций**: Отображение bounding box'ов с цветовой градацией в зависимости от уверенности (confidence).
*   **Навигация**: Удобное переключение между изображениями с помощью кнопок или стрелок клавиатуры (`←`, `→`).
*   **Рекурсивный поиск**: Автоматический сбор всех поддерживаемых изображений из выбранной папки и подпапок.
*   **Статистика**: Вывод количества найденных объектов и текущей позиции в логе консоли.

## Требования

*   Python 3.8+
*   Библиотеки:
    *   `ultralytics`
    *   `opencv-python`
    *   `Pillow`
    *   `tkinter`

## Установка

1.  Установите необходимые зависимости:

```bash
pip install ultralytics opencv-python pillow
```

2.  Подготовьте модель:
    *   Поместите файл весов модели (например, `best.pt`) в директорию `../` относительно скрипта или измените путь `MODEL_PATH` в коде.

## Использование

1.  Запустите скрипт:

```bash
python main.py
```

2.  Нажмите кнопку **«Выбрать папку»** и укажите директорию с изображениями для тестирования.
3.  Используйте кнопки навигации или стрелки на клавиатуре для просмотра результатов.

## Конфигурация

Основные параметры можно изменить непосредственно в начале файла `main.py`:

| Параметр | Описание | Значение по умолчанию |
| :--- | :--- | :--- |
| `OBJECT_NAME` | Название класса объекта для отображения | `'proto_enemy'` |
| `CONFIDENCE_THRESHOLD` | Порог уверенности для фильтрации детекций | `0.5` |
| `MODEL_PATH` | Путь к файлу весов YOLO | `'../best.pt'` |

## Цветовая индикация

Цвет рамки зависит от уровня уверенности модели (`conf`):
*   **Красный**: Низкая уверенность (`conf <= 0.2`).
*   **Градиент (Красный → Зеленый)**: Средняя и высокая уверенность (`0.2 < conf <= 1.0`).
